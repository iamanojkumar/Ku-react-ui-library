# TextField

The TextField component is a versatile input field that allows users to enter and edit text. It supports various styles, sizes, and states to accommodate different use cases.

## Features

- Multiple variants: outlined, filled, and subtle
- Different sizes: small, medium, and large
- Support for labels, helper text, and error messages
- Icons and affixes (prefix/suffix)
- Disabled and error states
- Dark theme support
- High contrast mode support
- Reduced motion support

## Usage

```tsx
import { TextField } from '@ku/react';

function MyForm() {
  return (
    <TextField
      label="Username"
      placeholder="Enter your username"
      helperText="Must be at least 3 characters"
    />
  );
}
```

## Accessibility

The TextField component follows WAI-ARIA guidelines for form inputs:

- Uses semantic HTML elements (`<label>`, `<input>`)
- Supports aria-label and aria-describedby
- Provides visual feedback for focus and error states
- Works with screen readers
- Supports high contrast mode
- Respects reduced motion preferences

## Best Practices

1. Always provide a label for the input field
2. Use helper text to provide additional context when needed
3. Display error messages when validation fails
4. Choose the appropriate variant based on the context
5. Use consistent sizing within a form
6. Ensure sufficient color contrast for accessibility
7. Test with screen readers and keyboard navigation
